2. 一维数组和二维数组名和&arr数组名的含义

   + 一维数组和二维数组相同
     + 数组名代表首元素的地址，即第一个元素的地址
     + &数组名，是数组的地址

3. 一维数组和二维数组名的解引用

   * 一维数组
     + *arr：解引用出来是首元素的地址，==是地址不是值==
     + **arr：二级解引用出来才是首元素的值
   * 二维数组：
     + *arr：解引用出来是首元素的地址，首个一维数组的地址，即arr[0]
     + **arr：解引用出来是首个一维数组首元素的地址
     + ***arr：解引用出来时首个一维数组首元素的值
